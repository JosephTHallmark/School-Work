PCA core
	A mathematical core written in C#, uses Principal Component Analysis to preform similarity analysis. Written solo
	
PCA App
	A mobile application that uses the PCA core. Allows a user to take pictures of subjects, select areas information to then create a basis set. Then other users can use this basis for single pictures to be tested against. Written in C# and Xamarin. Written with a group during the software development course. Written for a professor in the chemistry department.
